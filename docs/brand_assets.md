# HELLA OPS 브랜드 자산 가이드

## 1. 공식 로고
- **파일명**: `hella.png`
- **물리적 위치**: `public/hella.png`
- **앱 내 참조 경로**: `/hella.png`

## 2. 사용 원칙 및 위치
로고는 시스템의 신뢰성과 브랜드 정체성을 유지하기 위해 아래 지정된 위치에 사용해야 합니다.

| 사용 위치 | 목적 | 스타일 가이드 |
| :--- | :--- | :--- |
| **로그인 화면** | 사용자 첫인상 및 브랜드 각인 | 중앙 상단 배치, 적절한 여백 유지 |
| **관리자 대시보드** | 시스템 소속감 제공 | 상단 네비게이션 바(LNB) 좌측 상단 |
| **사원 모바일 홈** | 고정 헤더 영역 | 헤더 좌측 또는 중앙 배치 |
| **공식 문서 (PDF)** | 문서 공신력 확보 | 리포트 우측 상단 또는 표지 하단 |

## 3. 주의 사항
- 로고의 파일명(`hella.png`) 및 확장자를 임의로 변경하지 마십시오.
- 비율을 무시한 강제 Stretch 또는 색상 반전 사용을 금지합니다.
- 새로운 스타일 변형이 필요할 경우 관리자 승인 하에 `public/` 내에 추가 자산으로 관리하십시오.
